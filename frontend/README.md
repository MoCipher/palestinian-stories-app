# Palestinian Stories App - Frontend

Welcome to the Palestinian Stories App frontend! This application allows users to share their stories, photos, and videos, while also providing an admin interface for managing content.

## Features

- **User Authentication**: Users can sign up and sign in to share their stories.
- **Story Submission**: Users can post stories with accompanying photos and videos.
- **Timeline Display**: A timeline view to showcase all submitted stories.
- **Admin Panel**: Admins can manage users and stories effectively.

## Technologies Used

- **Frontend Framework**: React
- **TypeScript**: For type safety and better development experience
- **CSS**: For styling the application

## Getting Started

### Prerequisites

- Node.js and npm installed on your machine.

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd palestinian-stories-app/frontend
   ```

2. Install dependencies:
   ```
   npm install
   ```

### Running the Application

To start the development server, run:
```
npm start
```

The application will be available at `http://localhost:3000`.

## Folder Structure

- **public/**: Contains the main HTML file.
- **src/**: Contains all the source code for the application.
  - **assets/**: Contains styles and other assets.
  - **components/**: Contains reusable React components.
  - **pages/**: Contains page components for routing.
  - **App.tsx**: Main application component.
  - **index.tsx**: Entry point for the React application.
